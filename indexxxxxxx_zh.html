为什么 HTML 比 Markdown 更有效？

Markdown

• 无法表示空间关系
• 表格功能简陋
• 无交互能力
• 排版能力有限

HTML

• SVG 图表、流程图
• 表单验证、动画、开关
• Diff 高亮、行号、注释
• 精确控制字体间距
